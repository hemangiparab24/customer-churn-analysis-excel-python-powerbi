<h2><a class="anchor" id="overview"></a>Overview</h2>

This project focuses on analyzing customer churn in a telecommunications company. The dataset was cleaned and preprocessed using Excel and Python, followed by exploratory data analysis (EDA) to identify factors influencing customer churn. Insights were visualized using an interactive Power BI dashboard.

---
<h2><a class="anchor" id="business-problem"></a>Business Problem</h2>

Customer churn directly impacts revenue and growth in the telecom industry.  
This project aims to:
- Calculate the overall customer churn rate
- Identify customer segments with higher churn risk
- Analyze the impact of contract type, tenure, payment method, and charges on churn
- Support data-driven customer retention strategies

<h2><a class="anchor" id="final-recommendations"></a>Final Recommendations</h2>

- Promote long-term contracts to reduce churn
- Offer retention plans for new and low-tenure customers
- Improve payment methods to lower churn risk
- Target high monthly charge customers with personalized offers
